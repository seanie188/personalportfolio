## 📌 Project Concept
This website serves as a comprehensive **Main Hub** designed to demonstrate our foundational learning in web development. It is structured as a central gateway that interconnects our technical assets, individual identities, and communication channels.

### **The Hub Architecture:**
1.  **Repository Integration:** A direct link to our source code, showcasing version control and collaborative development.
2.  **Portfolio Network:** A collection of individual pages representing the unique skills and progress of each team member.
3.  **Communication Portal:** A dedicated "Contact Us" section to simulate real-world user interaction and feedback loops.

## ⚙️ How the Website Works
The site is built using a **Modular Navigation System**. 

* **The Index (index.html):** Acts as the Router. It uses relative pathing to connect to individual `.html` files for each member.
* **The Social Sidebar:** Provides persistent access to the GitHub Repository for code transparency.
* **The Contact Section:** Uses form elements and anchors to facilitate user reach-out.
