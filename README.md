## 📌 Project Concept
This website serves as a comprehensive **Main Hub** designed to demonstrate our foundational learning in web development. It is structured as a central gateway that interconnects our technical assets, individual identities, and communication channels.

### **The Hub Architecture:**
1.  **Repository Integration:** A direct link to our source code, showcasing version control and collaborative development.
2.  **Portfolio Network:** A collection of individual pages representing the unique skills and progress of each team member.
3.  **Communication Portal:** A dedicated "Contact Us" section to simulate real-world user interaction.

---

## 🛠️ Technologies & Tools Applied

#### **1. Core Frontend Development**
* **HTML5:** Semantic structure for the "Main Hub" and individual portfolio pages.
* **CSS3:** Custom styling, Flexbox layouts, hover transitions, and navigation bar positioning.

#### **2. Version Control & Collaboration**
* **Git & GitHub:** Used for tracking changes, team collaboration, and codebase management.

#### **3. Hosting & Automation**
* **GitHub Pages & Actions:** Utilized for live web hosting and automated deployment workflows.

#### **4. Design & Assets**
* **Photopea / Canva:** Used for graphic design, asset rasterization, and UI element creation.
